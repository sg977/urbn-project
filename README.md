## General info
This project is based on React.js, Google APIs and Antd https://sg977.github.io/urbn-project/
	
## Technologies
Project is created with:
* react
* react-dom
* react-router-dom
* react-scripts
* google-map-react
* Google API
* gh pages
	
## Setup
To run this project, install it locally using npm:

```
$ clone the repo
$ cd project
$ npm install
$ npm start
```
